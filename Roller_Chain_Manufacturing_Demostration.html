Roller Chain Manufacturing Demonstration

A pubilcly hosted demo that I built in the ICONICS platform.
Expanded ICONICS software into different market verticals.

Project Summary

When I first started working at ICONICS, I was impressed with the company's public-facing demonstrations. These demonstrations covered multiple industries and showed the power of the ICONICS platform. However, I noticed that one industry was missing from the public-facing demonstrations. The missing industry was discrete manufacturing. This gap inspired me to draw on my previous experience as a manufacturing engineer and create a new public-facing demonstration for the discrete manufacturing vertical.

Demo Objectives

I wanted to create a demonstration that mirrored my experience working in a chain manufacturing plant. I brainstormed what would be important to an operator/engineer at the plant. I came up with a handful of key aspects:
• Equipment Performance
• Real-time and Historical Analysis
• Intuitive Navigation

UI/UX Design

A large part of any demonstration system is creating a user interface that is not only visually attractive but also provides an intuitive user experience. I built the UI/UX for this project using ICONICS native dashboarding tool. ICONICS has featured this chain manufacturing demonstration in presentations leading to several software purchases and paid customer engagements.

Techincal Insights

An appealing visual is only half the battle. To create the data powering this project I had to refer to my previous knowledge as a manufacturing engineer to construct realistic graphics and charts. This required extensive use of Microsoft SQL Server and the ICONICS Suite of tools. Once the data was created, I had to use ICONICS SQL-like language to display and dynamically manipulate the underlying production data.
