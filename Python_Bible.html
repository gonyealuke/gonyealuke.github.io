The Python Bible

The Udemy Certifcation course I took on Python Programming.
Self-identifying my growth areas and expanding my techincal toolset.

Why Python?

Working at ICONICS exposed me to a world of software that I knew nothing about. I went from not being able to open my File Explorer to being a product expert for an industry leading SCADA software. I wanted to formalize this technical skillset with a formal training class. Python, being the leading open-source programming language was the perfect place for me to start.

Project Highlights

The Python Bible has 9 hours of video lessons. These lessons include 11 coding projects that I completed in addition to the lectures. These projects reinforced core concepts such as data structures, loops, functions, and object-oriented programming to expose me to real-life applications of Python coding.

Lessons Learned

Taking the Python Bible course cemented knowledge that I have picked up along the way using the ICONICS software. The two things I took the most value from was learning more about using loops in code and the concept of object-oriented programming. I had prior knowledge of both subjects but did not have a formal understanding of how powerful these tools really are in the hands of a developer.

What's next?

I do not see myself making the shift to becoming a developer, however I do think having a better understanding of coding principles is a valuable asset for any Sales Engineer. The lessons I learned in the Python Bible will make me better suited to help customers fulfill their technology needs.
